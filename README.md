# 🛒 Online Shoppers Purchasing Intention — Classification Project

A supervised machine learning project focused on predicting whether an online visitor will complete a purchase (**Revenue: True/False**) based on session behavior and engagement metrics.

---

## 📌 Project Overview

E-commerce platforms generate large volumes of user interaction data. Understanding which visitors are likely to convert is crucial for:

- Improving marketing efficiency  
- Personalizing recommendations  
- Optimizing conversion rates  
- Increasing revenue  

This project builds multiple classification models to predict purchase intention using real-world session data.

---

## 🎯 Objectives

### 🔹 Primary Objective
Develop a supervised ML classification model to predict online shopper purchase intention.

### 🔹 Specific Objectives

- Analyze browsing behavior and session duration
- Study the impact of visitor type and traffic source
- Evaluate temporal factors such as month and weekend
- Identify high-intent users for targeted marketing
- Support data-driven business decisions

---

## 📊 Dataset Information

- **Dataset:** Online Shoppers Purchasing Intention Dataset  
- **Source:** Kaggle  
- **Total Records:** 12,330 sessions  
- **Total Features:** 18  
- **Learning Type:** Supervised Learning  
- **Problem Type:** Classification  
- **Target Variable:** Revenue (Purchase: Yes/No)

### 🧩 Key Features

**Behavioral Metrics**

- Administrative pages & duration  
- Informational pages & duration  
- Product-related pages & duration  
- Bounce Rate  
- Exit Rate  
- Page Value  

**Session Context**

- Special Day proximity  
- Month  
- Weekend indicator  

**User & Technical Information**

- Operating System  
- Browser  
- Region  
- Traffic Type  
- Visitor Type
